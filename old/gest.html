HAND GESTURE RECOGNITION

HAND GESTURE RECOGNITION:



Introduction

This project aims at using image processing to recognize and display the letters made using hand gestures. Image processing is a form of signal processing in which the input is an image, such as a photograph or video frame. The output of image processing may be either an image or, a set of characteristics or parameters related to the image.
The steps involved in achieving this project are :-
• Video acquisition
• Use of filters
• Comparing the acquired image with the database
• Displaying the result


Video Acquisition

The first stage of any vision system is the Video acquisition stage. After the Video has been obtained, various methods of processing can be applied to the Video to perform the many different vision tasks required. A video of 50 frames is captured in our project.



Use of filters

Filters are used on the acquired video in order to obtain only the movement of the hand so that the letter depicted by the hand movement can be recognized. The filters are applied to all the frames and the frames are added up to form the required image.


Comparing the processed image with the database

The processed image is then compared to all the letters in the database one-by-one and the best result is then displayed on the screen. This process is done with the help of matlab software. Hence with the help of Matlab software and the concepts of image processing the hand gestures can be recognized and the results can be displayed. In this project we have used image processing to recognize alphabets. It can be used to detect any kind of hand gesture provided a database is created for the detection of those hand gestures
